# Python 算法实践：八皇后问题 (N-Queens)

本项目包含了一个基于**回溯算法**的高效八皇后问题求解器。通过位运算（Bit Manipulation）优化，显著提升了搜索效率。

## 🚀 核心特性
- **位运算优化**：利用位掩码将安全检查复杂度从 $O(N)$ 降至 $O(1)$，极大提升了在大规模棋盘下的运行速度。
- **可视化输出**：精美的终端棋盘渲染，使用 Emoji（👑, ⬜, ⬛）直观展示皇后布局。
- **面向对象设计**：结构清晰，易于扩展，支持从 4 皇后到任意 N 皇后的求解。

## 🛠️ 如何运行
确保你已安装 Python 3.6 或更高版本，然后在命令行执行：

```bash
python "回溯算法（八皇后问题）.py"
